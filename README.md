# AI Career OS

AI Career OS is a Python-based career intelligence tool designed to help professionals manage career profiles, calculate career scores, analyze salary expectations, and identify skill gaps for target roles.

## 🚀 Features

- Create career profiles
- Save profiles using JSON
- View saved profiles
- Edit profiles
- Calculate career scores
- Analyze salary expectations
- Analyze skill gaps by role
- Modular Python architecture
- Git and GitHub version control

## 🧠 Tech Stack

- Python
- JSON
- Git
- GitHub
- Modular architecture
